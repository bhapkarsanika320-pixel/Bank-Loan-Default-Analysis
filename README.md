# Bank-Loan-Default-Analysis
## Bank Loan Default Prediction using Python, SQL, Machine Learning and Power BI

## Overview
Analyzed loan records to predict default risk using Python, SQL, Machine Learning and Power BI.

## Tools used
- Python (Pandas, Seaborn, Scikit-learn)
- MySQL
- Power BI
- Machine Learning (Logistic Regression, Random Forest)

## Key findinds
- Only 6% customers defaulted
- Young customers (under 30) default the most (10.4%)
- credit utilization is the predictor of default
- Random Forest achieved 93.87% accuracy
